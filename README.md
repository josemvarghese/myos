# Mayos - Jose  - Online shop API

Build an online shop API with the following features:
- Product API:
  - API to list all the products with
  - search: by title and description
  - sort: by price
- Checkout API:
     - API to create an order
## Tech Used

- NodeJs
- Typescript
- ExpressJs
- Postgres
- Docker 

## To Run 

```sh
cd myos
docker-compose build
docker-compose up
```
